-----------------
Toasty.js
-----------------

DEMO:
https://dl.dropbox.com/u/21334841/toasty.js/demo.html

SUMMARY:
* A jQuery plugin that shows the Mortal Kombat toasty guy!

REQUIRES:
* jQuery

HOW TO USE:
* Add the toasty.js file, along with the audio and images, to your site's root directory.
* Call the toasty() method on the element that will be clicked to show the toasty guy.

EXAMPLE:
<a id="toasty" href="#">Ka-ching!</a>

<script>
  $("#toasty").Toasty();
</script>
