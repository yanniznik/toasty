# mk-toasty
Make the Mortal Kombat toasty guy pop out on your site or web app.

## Installation

```
$ npm install mk-toasty
```

## Build

```
$ npm run build
```

## Usage

**HTML**:
```html
<button class="toasty">Click me!</button>
```

**JS**:
```js
var Toasty = require('toasty!');

var toasty = new Toasty('.toasty');
toasty.add();
```
